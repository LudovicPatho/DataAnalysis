# Courses from Dataquest
This folder gathers the various exercises and projects made through the courses:
- [Python Programming Beginner](Dataquest%2001%20Python%20Programming%20Beginner/): Python basics, lists, functions, files, loops, dictionnaries, debugging errors, booleans, if statements
- [Python Programming Intermediate](Dataquest%2002%20Python%20Programming%20Intermediate/): list comprehensions, error handling, classes, modules, variable scopes, regular expressions, dates
- [Data analysis with Pandas Intermediate](Dataquest%2003%20Data%20analysis%20with%20Pandas%20Intermediate): numpy, pandas, pivot tables, missing data
